# De la tradición al Remix

Web estática para el proyecto educativo "De la tradición al Remix: músicas que viajan y se transforman".

## Desarrollo local

```bash
python3 -m http.server 8000
```

Después abre `http://localhost:8000`.

La web usa solo HTML, CSS y JavaScript. Las fotos originales se mantienen intactas y las copias web se guardan en `assets/media/photos`. Los vídeos se reproducen desde sus carpetas originales y usan miniaturas generadas en `assets/media/posters`.
